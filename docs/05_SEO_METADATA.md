## SEO STRATEGY SUMMARY

### Keyword Strategy Hierarchy

**Tier 1 - Broad (Homepage, Hub Pages)**

- Bali tours
- Bali experiences
- Things to do Bali

**Tier 2 - Destination (Destination Pages)**

- [Destination] tourism
- Best things to do [Destination]
- [Destination] tour

**Tier 3 - Activity Specific (Product Pages)**

- [Activity] tour Bali
- [Activity] experience
- Best [activity] Bali

**Tier 4 - Ultra Long-tail (Product Pages)**

- [Activity] with local guide Bali
- Authentic [activity] experience Bali
- [Specific detail] [activity] Bali

### Technical SEO Recommendations

**URL Structure:**

- Keep URLs under 75 characters
- Use hyphens to separate words
- Avoid special characters or symbols
- Include primary keyword in slug when possible

**Heading Hierarchy:**

- One H1 per page (brand name or main heading)
- H2 for main content sections
- H3 for subsections
- Never skip heading levels

**Internal Linking:**

- Each product page: 3-5 contextual links to related experiences
- Each destination page: 4-6 links to relevant activities
- Home page: Essential links to all major categories

**Image Optimization:**

- Use descriptive file names (mount-batur-sunrise-guide.jpg)
- Include alt text with relevant keywords
- Compress images to <200KB where possible
- Use modern formats (WebP when supported)

**Schema Markup:**

- Use LocalBusiness schema for local SEO
- Use Event schema for each experience/tour
- Use BreadcrumbList for navigation
- Use Review schema for testimonials

**Mobile Optimization:**

- All pages fully responsive
- Fast loading times (target <3 seconds)
- Touch-friendly buttons and CTAs
- Readable font sizes

## SEO MONITORING

### KPIs to Track

- Organic traffic to each destination page
- Average position for top keywords
- Click-through rate (CTR) from SERPs
- Conversion rate by source page
- Bounce rate by page type
- Time on page
- Pages per session

### Tools Recommended

- Google Search Console (track rankings, impressions, clicks)
- Google Analytics 4 (traffic, behavior, conversions)
- SEMrush or Ahrefs (rank tracking, competitor analysis)
- Screaming Frog (technical SEO audits)
- PageSpeed Insights (performance monitoring)

### Quarterly Audit

- Review keyword rankings
- Identify underperforming pages
- Check for broken internal links
- Analyze competitor strategies
- Optimize underutilized keywords
- Update outdated content
